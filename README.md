# design-patterns

Estudos de Design Patterns

Inicializando projeto node.js

```
npm init -y
```

Instalando typescript com desenvolvimento

```
npm i typescript -D
```

Configura o typescript com o `tsconfig.json`

Compilar arquivos TS para Js. Será criado uma pasta dist com um arquivo js. Caso precise. Não será utilizado no projeto

```
npx tsc
```

Executa o typescript como linguagem interpretada

```
npm i -D ts-node
```

Instala as configurações de lint e prettier

```
npm i -D eslint @typescript-eslint/eslint-plugin @typescript-eslint/parser prettier eslint-plugin-prettier
```
